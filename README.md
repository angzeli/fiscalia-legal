# Fiscalia Legal 🌿

This repository hosts the public legal and support pages for **Fiscalia** — a local-first personal finance app for tracking spending, budgets, and recurring expenses on iPhone 📱

- `privacy.html` — Privacy Policy 🔒
- `terms.html` — Terms of Use 📄

## About Fiscalia 💸

Fiscalia is designed to be simple, private, and local-first.

- no account required
- no cloud sync
- no ads
- no tracking
- user data stored locally on-device only
